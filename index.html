Machine Failure Prediction using ML Algorithm

Main project Feb to June 2023 MNMJEC

Developed machine lifespan and failure prediction using Python and advanced ML algorithms.

Project Description:

Machine Failure, or Equipment Failure, is any event in which a piece of industrial machinery underperforms, whether entirely or partially, or stops functioning in the way in which it was intended to. This project, conducted from February to June 2023 at MNMJEC, focuses on predicting machine failure using advanced Machine Learning (ML) algorithms.

The algorithms employed in this project include logistic regression, support vector machine, decision tree, and naive bayes, offering a comprehensive approach to failure prediction.

Keywords: Machine Failure, Machine Learning, classification

Abstract:

Machine failure in industrial settings can lead to significant consequences, including loss of asset availability, deviation from standard procedure, and financial losses. This project addresses these challenges by utilizing data mining and machine learning techniques to predict and prevent machine failure.

The project involves the comparison of different ML algorithms, such as logistic regression, support vector machine, decision tree, and naive bayes, to determine the most effective model for predicting machine failure.

Description:

The proposed system aims to build a machine learning model for predicting machine failure. It involves collecting and preprocessing the dataset, analyzing and training the data using machine learning algorithms, and deploying the best model for prediction. The advantages of the proposed system include the implementation of machine learning, accuracy prediction, and live testing.

Preparing the Dataset:

This dataset contains 100010 records of data, with the target column having three classifications. The dataset undergoes data pre-processing, including validation techniques, to ensure an unbiased evaluation of the machine learning model.

Data Pre-processing:

Validation techniques in machine learning are employed to get the error rate of the ML model. This involves dealing with samples of data that may not be a true representative of the population. The dataset is validated, missing values are addressed, and unwanted data is removed.

Support Vector Machine:

Support Vector Machine (SVM) is a popular supervised learning algorithm used for both classification and regression. In this project, SVM is applied to create the best decision boundary for segregating n-dimensional space into classes, aiding in machine failure prediction.

Decision Tree Classifier:

Decision Tree is a versatile supervised learning technique used for classification problems. It involves creating a tree-structured classifier where internal nodes represent features, branches represent decision rules, and leaf nodes represent outcomes. Decision trees are employed to enhance machine failure prediction.

Conclusion:

The analytical process started from data cleaning and processing, missing value handling, exploratory analysis, and concluded with model building and evaluation. The project provides a robust solution for predicting machine failure, leveraging the power of machine learning algorithms.

The accuracy achieved in live testing demonstrates the effectiveness of the proposed system in predicting machine failure and preventing potential industrial setbacks.

Voice Controlling Robot

Embark on an interactive journey with our innovative "Voice Controlling Robot" project, crafted during the mini project period from March to May 2022 at MNMJEC. This project introduces the revolutionary Burnon app, connecting an Arduino board and Python language, compiled by Proteus software. The app provides an intuitive interface for controlling the robot through voice commands, exploring the future of human-machine interaction with this cutting-edge mini project.

Group Members:

  • Dharanidharan K.C
  • Sruthin S
  • Gokul Raj S

Description:

In this scientific report, a system is developed focusing on using human voice to operate a robot. The voice-controlled robot is an example of controlling the motions of a simple robot using commonly used voice commands. An android app is utilized to transmit human commands to the microcontroller. The UART protocol is used to connect a controller to a Bluetooth module. The android app receives the speech, processes it, and transforms the voice to text.

Step 1: The microcontroller processes this text and takes appropriate action to control the robot.

Step 2: The goal is to create a robotic car that can be controlled by human voice for basic movements like going forward and turning left or right.

The ATmega Arduino board is the Hardware Development board utilized in this project. The software is written in the Arduino IDE with Embedded C. Hardware has been installed, and software has been ported.

After a large number of tests, it was concluded that there is a simple and effective technique to manipulate robots using human voice. This ergonomic technique for robotic application simplicity provides excellent assistance while executing several duties.

Arduino Based Voice Controlled Robot Vehicle

Introduction:

Our goal is to create a robot car that can be operated by a person's voice command. This system, a prototype of our design, utilizes an Android application with a microcontroller. Bluetooth technology facilitates the connection between the application and the robot. The commands are passed via the channel to the module, which receives them. The voice-controlled robotic vehicle (VCRV) listens to and responds to the user's commands, requiring accent training for the gadget to grasp the commands given.

Principles:

Arduino is a microcontroller or a little computer. It lacks the necessary processing power and capabilities to perform speech recognition. However, the PCB board features a connection for connecting a Bluetooth module. This allows the Arduino to connect to other gadgets wirelessly. The Android app conducts "Speech to Text" operations using Google API and then sends commands to Arduino through Bluetooth after getting the text.

Improvements:

The project "Voice Controlled Robotic Vehicle" has numerous uses both now and in the future. In the future, improvements can be added to make it more effective. The project has a wide range of applications, including military, home security, rescue missions, industry, and medical support. The project is simple to implement and advantageous to human life, especially for monitoring and assisting disabled persons.
